How to Run the Application?



Step 1: Start the Backend Server

1. Open your terminal or command prompt.

2\. Navigate to the project directory where the server.js file is located.

3\. Execute the following command to start the server: node server.js



Step 2: Launch the Client (Frontend)

1. Open your file explorer and navigate to the public folder inside the project directory.

2\. Locate the index.html file.

3\. Open the file using your preferred web browser (e.g., Google Chrome, Firefox, or Edge).





Prerequisites: 

Ensure you have Node.js installed on your machine before running the server command.



Troubleshooting: 

If the page does not load correctly, check your terminal to ensure the server is still running without errors.
